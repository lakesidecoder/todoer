
# Todoer

This is a Todo app built with Vue3 and local storage for practice.


## Demo

https://todoer-kz40.onrender.com
## Features

- Add task
- Edit task
- Delete task
- Data persistence


## Deployment

To deploy this project run

```bash
  npm i && npm run build
```
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run serve
```
